## IOT HTTP Gateway
 Gateway that Consumes Location data from a mobile phone and forwards it via HTTP Post to an IOT platform. 
### Prerequisites
    1. NodeJS
    2. Docker (Optional)

### Clone the repository

```bash
git clone https://github.com/munyiwamicha/iotgateway.git
```
### Installation

```bash
npm install
```
